스트리밍 성과발표회

2020.11.20.

PM 13:00~18:00 FRI

문화체육관광부와 한국콘텐츠진흥원은 홍릉 콘텐츠문화광장에서
온라인 ‘무한한 가능성으로 세상을 깨우다’라는 컨셉으로
2020 콘텐츠 창의인재 양성사업 통합 성과발표회는
COVIED-19라는 예상 못한 폭풍이 불어오는 어려운 현시점에서도
우리의 '휘장'은 멈추어 서거나, 포기하지 않고 창의인재들의 의지와 노력으로
어려움조차 기회의 발판으로 삼아 한계를 뛰어넘어 다양한 프로젝트를 수행했습니다.
첨단 기술이 콘텐츠와 결합된 다양한 형태의 결과물들을 한눈에 확인할 수 있습니다.
