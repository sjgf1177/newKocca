프로그래밍 개론
2005.12.24
2005.12.24
프로그래밍1은 게임개발플랫폼에 제약을 받지 않는 C언어를 학습 목적으로 한다.
C언어는 구조화된 언어로 게임개발뿐 만아니라 각종 응용프로그램 개발에도 사용되어 학습자에게 프로그래밍의
기본 개념을 이해하고 수행원리를 습득하는데 많은 도움을 준다.
또한 모든 게임 프로그래밍 교과목의 근간이 되기 때문에 학습자는 반드시 본 과목의 모든 과정을 완전히 이해하고
실습을 통해 숙지하여야 한다.
- 프로그래밍의 기본 개념 이해와 수행원리 습득
- 과제를 통한 C언어 문법 숙지
- 미니 프로젝트를 수행으로 C언어 정복
