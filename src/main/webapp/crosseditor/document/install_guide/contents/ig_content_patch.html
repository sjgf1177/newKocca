패치 방법

크로스에디터는 패치파일만 독립적으로 사용할 수 없습니다.

크로스에디터 패치파일을 받은 경우 아래의 방법으로 패치를 진행합니다.

ASP, ASP.NET, PHP

  1. 크로스에디터 패치 파일을 크로스에디터 설치 폴더에 덮어쓰기 합니다.
  2. 관리자 페이지에 접속하여 로그인을 하면 패치가 완료됩니다.

JSP

  1. 크로스에디터 패치 파일을 크로스에디터 설치 폴더에 덮어쓰기 합니다.
  2. 서버가 JSP파일을 reload 할 수 없는 경우 서버를 재 시작합니다.
  3. 관리자 페이지에 접속하여 로그인을 하면 패치가 완료됩니다.

SERVLET

  1. "패치 파일 폴더/websource" 폴더를 제외한 나머지 크로스에디터 패치 파일들을 크로스에디터 설치 폴더에 덮어쓰기 합니다.
  2. 서버를 종료하고 "패치 파일 폴더/websource/servlet/(classes or lib)" 폴더들을 "$WebApplication_Root/WEB-INF/"에 덮어쓰기 한 후 서버를 재 시작합니다.
  3. 관리자 페이지에 접속하여 로그인을 하면 패치가 완료됩니다.
