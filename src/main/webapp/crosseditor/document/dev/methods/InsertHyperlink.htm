InsertHyperlink

현재 캐럿이 위치한 곳에 하이퍼링크를 생성합니다.

 

Syntax

    HRESULT InsertHyperlink (
            BSTR str,
            BSTR href
    );

Parameters

str

[in] 문서에 표시할 문자열을 입력합니다.

입력 캐럿이 하이퍼링크에 있거나, 블럭이 설정되어 있을 경우에는 str 은 무시되며, 그 외의 경우에만 적용됩니다. 또한 str 이 없는 경우는 href 가 사용됩니다.

href

[in] 하이퍼링크 주소를 입력합니다.

 

Return Values

없음

 

Remarks

메소드는 Main class 호출 후 에디터가 로드된 다음 실행해야 정상적으로 실행됩니다.

str 값을 입력 하지 않은 경우 하이퍼링크의 text 값은 href 값을 사용합니다.

 

Sample Codes

JavaScript

var object = new NamoSE("test");
object.editorStart();
object.InsertHyperlink("Namo","http://www.namo.co.kr");
