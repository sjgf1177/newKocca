doCommandv3.0 or higher

지정한 명령을 실행합니다.

 

Syntax

    void doCommand (
     BSTR cmdstr,
     LPVOID cmdElem,
     BSTR newVal
);

Parameters

cmdstr

[in] 실행할 명령 키워드를 입력합니다. (해당 키워드 CreateToolbar 참조)

cmdElem

[in] 명령을 수행한 버튼 및 이미지 객체 (해당 객체가 존재하는 위치에 따라서 플러인의 위치가 결정됩니다.)

Paramstr

[in] 명령 키워드의 인자들을 입력합니다. (인자가 필요한 명령에 대해서만 설정이 됩니다. 필요하지 않는 명령어일 경우는 해당 인자를 설정하지 않습니다.)

fontname

폰트 종류 ex) "돋움"

fontsize

폰트 크기 ex) "9pt"

lineheight

줄 간격 ex) "100"

formatblock

제목 종류 ex) "h1"

 

Return Values

없음

 

Remarks

메소드는 Main class 호출 후 에디터가 로드된 다음 실행해야 정상적으로 실행됩니다..

 

Sample Codes

JavaScript

var ele = document.getElementById('exec');
object.UseDoCommand = true;
object.doCommand('newdoc', ele);
object.doCommand('fontname', ele, '돋움');
